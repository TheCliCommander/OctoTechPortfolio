Linux System Administration

I absolutley love Linux! I spent over 100 hours just taking video courses when I first met Tux. I started my adventures with python on a Linux distro and my first few projects were ethical hacking tools like webcrawlers, vulnerability scanners, backdoor and listener etc. You can find all of the code for these projects on my github:TheCliCommander I am well versed in setting up permissions for users, creating new users, setting passwords and switching to other users, using the PS command to identify processes and kill them when necessary and can utilize Top to do the same.
